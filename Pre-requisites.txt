1) Ensure that ssh_certificates are placed in ./rpi_monitor/mirpi/ssh_certificates
Public Key Filename: id_rsa2.pub
Private Key Filename: id_rsa2

2) Set Email credentials and other network configurations in ./rpi_monitor/mirpi/cnst.py

3) cd to ./, and deploy with the following line of code

(sudo) docker-compose up --build

On RPi, the build process takes about 3/4 hours on a 2Mbps connection. Mileage may vary.
#------
Once deployed, the Host Server will be accessible on the Host machinse port 80, ready for Hubs to interact with.
